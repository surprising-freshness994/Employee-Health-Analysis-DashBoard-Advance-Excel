## 🛠 How to update or add your own data

To analyze a new dataset using this system:

1. Format your data to match the columns in the raw Excel file.
2. Clear any errors or missing entries.
3. Copy and paste your cleaned data into the cleaned data file, replacing current entries.
4. Refresh pivot tables in the dashboard Excel file:
   - Click inside a pivot table.
   - Go to the **PivotTable Analyze** tab.
   - Click **Refresh**.

This will update all charts and summaries.

## 🔍 Understanding the Data Cleaning Workflow

The cleaning process addresses:

- Missing values replaced using average or modes.
- Date formats standardized to `DD/MM/YYYY`.
- Removal of duplicate entries.
- Correction of typos in department and health status fields.
- Fixing inconsistent labels (e.g., “Yes/No” vs “Y/N”).

You can open the `Data Cleaning Steps` sheet inside `Employee_Health_Cleaned.xlsx` for detailed notes.

## 📈 About the Dashboard

The dashboard uses **pivot tables** to:

- Summarize monthly health check participation.

- Track improvements in health KPIs over time.
- Visualize metrics with charts tied directly to pivot data.

Pivot tables let you quickly slice data by categories without writing formulas. The dashboard works fully offline inside Excel.

## ⚙️ Troubleshooting Tips

- Make sure Microsoft Excel is fully updated.
- If you see protected view warnings, click **Enable Editing** to work with files.
- If pivot tables do not update correctly, right-click inside the table and select **Refresh**.
- For performance, close other large applications while running Excel.
- If Excel shows data errors, check your source file formatting matches the template.
